# Canonical Tests

This folder contains test pages for canonical tag scenarios.

## What is a Canonical Tag?
A canonical tag tells search engines which version of a page 
is the "main" one — useful when you have duplicate or 
similar pages.

## Test Scenarios in This Folder
- Canonical pointing to a broken URL
- Canonical pointing to a valid URL
- Multiple canonical tags on one page
- Canonical via HTTP only
- Canonical with JavaScript

## Notes
- Do NOT modify existing canonical test files outside this folder
- Each file should have a clear test description in its frontmatter
